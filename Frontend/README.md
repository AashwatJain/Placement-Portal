# 🎓 NIT KKR Placement Portal

A modern, frontend-only placement management portal designed for **NIT Kurukshetra**. Built with **React** and **Tailwind CSS**, this portal streamlines the placement process for Students, Admins, and Recruiters with a clean and responsive UI.

> **Note:** This is currently a **Frontend Prototype**. It uses mock data and does not have a connected backend or active database yet.

---

## 🚀 Features

### 👨‍🎓 Student Module
* **Dashboard:** Real-time stats on applications (Applied, Shortlisted, Pending) and action items.
* **Profile Management:** Edit academic details, social links (GitHub/LinkedIn), and upload resume/transcripts.
* **Resume Vault:** Manage multiple resumes and view mock ATS analysis scores.
* **Placement Drives:** Browse companies with filters (Year, Type), view CTC details, and access preparation materials.
* **Interview Experiences:** Read archive experiences from seniors or share your own journey.
* **Application Tracking:** detailed status tracking of all applied drives.

### 👮‍♂️ Admin Module
* **Student Management:** View and manage student data.
* **Company Management:** Add new placement drives and update status.
* **Question Bank:** Manage the repository of interview questions.

### 💼 Recruiter Module
* **Dashboard:** Overview of hiring requirements.
* **Candidate Sorting:** Filter students based on CGPA and skills.

### 🗓️ Shared Features
* **Calendar:** Unified view of upcoming OAs, deadlines, and interviews.
* **Role-Based Access:** Seamless switching between roles (Student/Admin/Recruiter) for demo purposes.
* **Dark Mode:** Fully supported system-wide dark/light theme.

---

## 🛠️ Tech Stack

* **Framework:** [React](https://reactjs.org/) (via [Vite](https://vitejs.dev/))
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Routing:** [React Router DOM](https://reactrouter.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **State Management:** React Context API (Auth & Theme)
